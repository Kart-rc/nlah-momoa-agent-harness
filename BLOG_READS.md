# Engineering Blog Reading Tracker

A daily log of one interesting engineering article — system design, data engineering,
leadership, software engineering, architecture, or data observability. Each entry has
a deep-dive write-up plus a small visual app that explains the concept.

**Process**

1. Read this file first every day to avoid repeating a topic.
2. Pick one article from the source list.
3. Add a row to the table below (date, source, title, topic, link, folder).
4. Write notes + build a small visual app under `blog-reads/<YYYY-MM-DD>-<slug>/`.
5. Commit on branch `claude/aggregate-engineering-blogs-XaC0U`.
